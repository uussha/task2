# Internship Task 2

## Project Description
Recreated the Internship Task 1 document using the Nunjucks templating engine and Vite bundler while maintaining the original formatting and structure.

## Tech Stack
- **Templating Engine:** Nunjucks
- **Bundler:** Vite

## Installation

Install the project dependencies:

```bash
npm install
```

## Run the Project

Start the development server:

```bash
npm run dev
```

## Build the Project

Generate the production build:

```bash
npm run build
```

The compiled output will be generated in the `dist/` folder.
